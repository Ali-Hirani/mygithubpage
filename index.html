About Me!

My name is Ali Hirani and I am in my first year of Computer Engineering at the University of Waterloo. I am strongly self-motivated in learning all things computing whether it's object oriented programming in C#, mobile app development for the Android platform with Java and Android Studio, Web development with HTML and CSS or the advanced use of various Linux distributions such as CentOS, Ubuntu and OpenSUSE.

I strongly believe that these sorts of topics can be best learned through practice and independent developement. As a result, I put a team of amateur programmers, such as myself, together and decided to build a console application using the object oriented programming techniques we learned through a C# Fundamentals Course.

We collaborated on the project through a Git repository service known as GitHub. The source code for the project can be found here, and the application itself can be downloaded here.
